Redistricting

Last graph file change: September 12th, 2011

These graphs represent US states. They are used for solving the redistricting problem. All data have been provided by Will Zhao. As stated on the project website, "(t)he nodes are Census2010 blocks. Two nodes have an edge linking them if they share a line segment on their (blocks') borders, i.e. rook-style neighboring. The nodes' weights are the POP100 variable of Census2010-PL94." For more information visit the project website and take a look at the README file.

Please use the command wget -r -np https://www2.informatik.hu-berlin.de/~wwwmaks/dimacs10/redistrict/ or something similar to download the files.
