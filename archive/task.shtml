Computational Task Graphs of Streaming Applications

Last update: August 12, 2011

A graph generator for computational task graphs of streaming applications has been contributed by Deepak Ajwani. More details can be found in the corresponding paper: Deepak Ajwani, Shoukat Ali, and John P. Morrison. Application-agnostic generation of synthetic task graphs for stream computing applications. Technical Report RC25181, IBM Research Reports, 2011.

For the graphs that were generated for test purposes we started at 20,000 vertices and increased |V| by a factor of four every time until |V| = 5,120,000. Deepak's code was changed to use a fixed seed, which allows you to reproduce our graphs. Check the md5sums. The number of nodes are used as the prefix in the file names.
