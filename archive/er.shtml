Erdös-Rényi Graphs

Last update: July 26, 2011

These random graphs have been generated according to the Erdös-Rényi G(n,p) model with p = 1.5 ln n / n. Download the generator if you want to avoid downloading the graphs. Simply use ./compile to compile the source code and run the script ./create-graphs-gnp.sh (requires 64-bit Linux, but should be customizable to other 64-bit platforms). Use the md5sums.txt file to check the validity of the compressed files. Please let Henning Meyerhenke know if you encounter any problems.
